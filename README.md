A python program that uses pygame and numpy to get user drawings and turn them into png files.

These png files are then analysed by an AI which allow it to make an accurate prediction as to 
what the number drawn on screen is

Unfortunetly, could not get a demo working in replit because of compatability issues, if you would 
really like to try the game out, you can download the zip and run the 'main.py' file.

If a config module not found error pops up, then copy and paste the code from the config files into 
the main and grid files while deleted the line that says 'from config import *'
